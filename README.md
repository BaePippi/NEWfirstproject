## 프로젝트 소개

<p align="justify">
노티드 도넛은 마케팅은 잘되었지만 공식 홈페이지가 없었습니다.
<br>
노티드가 속해있는 회사의 홈페이지에서 노티드에 대한 소개가 나와있지만 그것으로 노티드의 아이덴티티를 보여주기 어렵다고 판단하여 프로젝트를 진행하게 되었습니다.
</p>

<br>
